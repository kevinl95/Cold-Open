ColdOpen settings

These settings decide where aggregate-only class summaries are sent. No interaction events, session IDs, or retry history are stored here.

For this prototype, use your Pages URL followed by /api/class-summaries.

Separate approved sites with commas. ColdOpen collects timing-only events only on these domains; saving asks Chrome for permission for each one.

Use 3–128 letters, numbers, hyphens, or underscores. This is an opaque label, not a student identifier.
